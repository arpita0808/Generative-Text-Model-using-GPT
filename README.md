# Generative Text Model using GPT-2

An AI-powered Generative Text Model that generates coherent and context-aware text from user prompts using GPT-2 Transformer and LSTM architectures. The project also includes an interactive Gradio interface with adjustable creativity levels.

## Features

- Prompt-Based Text Generation
- GPT-2 Transformer Integration
- LSTM Text Generation
- Multiple Generation Modes (Creative, Balanced, Precise)
- Adjustable Creativity using Temperature
- Interactive Gradio Web Interface
- Comparative Analysis between LSTM and GPT-2

## Technologies Used

- Python
- TensorFlow
- Keras
- PyTorch
- Hugging Face Transformers
- GPT-2
- LSTM
- Gradio
- NumPy

## Project Workflow

1. Prepare and preprocess the dataset.
2. Train the LSTM model for sequential text prediction.
3. Load the pretrained GPT-2 model.
4. Accept a user prompt.
5. Generate text using LSTM and GPT-2.
6. Adjust creativity using temperature.
7. Display generated text through the Gradio interface.

## Installation

Clone the repository

```bash
git clone https://github.com/yourusername/Generative-Text-Model-using-GPT-2.git
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the project

```bash
python app.py
```
